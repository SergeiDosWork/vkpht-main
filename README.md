## Состав модулей

* boot - main класс запуска приложения
* common - общие классы для проекта
* mod-common - (старый rostalent) модуль с основными системными сущностями и базовыми методами доступа к ним
* mod-tasksetting - движок бизнес-процессов tasksettings
* mod-orgstructure - базовые методы доступа к орг.структуре
